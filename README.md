# Project 3: Around The U.S.

## Description

This project was made for the Sprint 3-5 of the Triple10 software engineering course. It was made based off of a Figma layout and interpreted by me.

## Project Features

- Grid
- Flexbox
- Custom Font
- Media Queries
- Responsive design
- DOM integration
